# Company Handbook

## Rules of Engagement

### Communication Rules
1. Always be polite and professional in all communications
2. Respond to urgent messages within 1 hour
3. Flag any message containing "urgent", "asap", or "emergency" for immediate attention

### Financial Rules
1. Flag any payment over $500 for human approval
2. Never initiate payments without explicit approval
3. Log all transactions in /Accounting folder

### Task Processing Rules
1. Process all files in /Needs_Action folder
2. Move completed tasks to /Done folder
3. Create approval requests in /Pending_Approval for sensitive actions
4. Never delete files - archive them instead

### Privacy Rules
1. Never share sensitive information externally
2. Keep all credentials and tokens in .env files (never commit)
3. Respect confidentiality of all business communications
